# ClawCube Product Definition

## Brand Concept
ClawCube is a family of OpenClaw-ready hardware products designed to make self-hosted AI more accessible, practical, and desirable.

## Product Lineup

### 1. ClawCube Lite
- CPU: J4125
- RAM: 8GB
- Role: entry model
- Intended user: beginners, lightweight personal automation, messaging agents
- Positioning: cheapest way to get started with OpenClaw

### 2. ClawCube Core
- CPU: Intel N100
- RAM: 16GB
- Role: mainstream model
- Intended user: most users, developers, self-hosters
- Positioning: the default recommendation for OpenClaw

### 3. ClawCube NAS
- Role: AI + storage hub
- Intended user: families, creators, self-hosters, backup-conscious users
- Positioning: your private AI box plus personal cloud

### 4. ClawCube Ultra
- GPU: RTX 4090 48GB-class
- Role: high-performance model
- Intended user: serious builders, researchers, local model users, power users
- Positioning: local AI beast for advanced workloads

## Product Architecture Questions
- Are Lite/Core fanless or actively cooled?
- Is NAS based on x86 or ARM?
- Is Ultra a desktop tower, cube, or rack-like unit?
- What software comes preinstalled?
- Is OpenClaw the hero software, or part of a broader AI stack?

## Initial Messaging Direction
- “OpenClaw, ready to run.”
- “From personal agent box to local AI powerhouse.”
- “Purpose-built hardware for AI-native workflows.”
